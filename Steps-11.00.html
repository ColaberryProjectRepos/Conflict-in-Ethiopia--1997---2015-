Creating Dax 11

To Create a DAX Measure, follow these steps:

·        Select the table “ AllMeasures”

·     Right Click and select New Measure

·      Write the following DAX Measures on the space provide one at a

time and save

Deaths per Mth = DIVIDE([TotalDeath],DISTINCTCOUNT('Date'[YearMonthnumber]),0) + 0

%MOM =

VAR TotalDeathLM =

    CALCULATE([TotalDeath], DATEADD('Date'[Date],-1,MONTH))

RETURN

DIVIDE(([TotalDeath]-TotalDeathLM),TotalDeathLM)

  

%YOY =

VAR TotalDeathLY =

    CALCULATE([TotalDeath], DATEADD('Date'[Date],-1,YEAR))

RETURN

DIVIDE([TotalDeath]-TotalDeathLY,TotalDeathLY)

 

TotalDeath = SUM(MasterXLS[FATALITIES])

 

TotalDeath_LM = CALCULATE([TotalDeath], DATEADD('Date'[Date],-1, MONTH))

 

TotalDeath_LY = CALCULATE(SUM(MasterXLS[FATALITIES]), DATEADD('Date'[Date],-1, YEAR))
