Create Date Table and Sort 7

Step1

From Power Bi desktop Menu click Modeling then click New Table and then paste and replace 'Table =' by the following Dax expression to create the date table:


Date =
ADDCOLUMNS (
CALENDAR (Min('MasterXLS'[EVENT_DATE]), MAX(MasterXLS[EVENT_DATE])),
"Dates", FORMAT ( [Date], "MM/DD/YYYY" ),
"DateAsInteger", FORMAT ( [Date], "YYYYMMDD" ),
"Year", YEAR ( [Date] ),
"Monthnumber", FORMAT ( [Date], "MM" ),
"YearMonthnumber", FORMAT ( [Date], "YYYY/MM" ),
"YearMonthShort", FORMAT ( [Date], "YYYY/mmm" ),
"MonthNameShort", FORMAT ( [Date], "mmm" ),
"Month", FORMAT ( [Date], "mmmm" ),
"DayOfWeekNumber", WEEKDAY ( [Date] ),
"DayOfWeek", FORMAT ( [Date], "dddd" ),
"DayOfWeekShort", FORMAT ( [Date], "ddd" ),
"Quarter", "Q" & FORMAT ( [Date], "Q" ),
"YearQuarter", FORMAT ( [Date], "YYYY" ) & "/Q" & FORMAT ( [Date], "Q" )

)

Step2
Sort the date Table:
Select Month then
in column tools select sort by column then
choose Monthnumber to sort
